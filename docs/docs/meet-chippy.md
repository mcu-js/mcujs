# Meet Chippy

Chippy is the mcujs mascot: part helpful guide, part tiny hardware cheerleader. You'll see Chippy pop up in the docs when you need a quick hint or a friendly nudge.

## Key terms

- [Mascot](./glossary.md)

## Why Chippy exists

- To make embedded JavaScript feel welcoming
- To remind you that hardware can be playful
- To keep the docs human
